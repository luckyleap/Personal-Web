Zero-shot Recognition

This is a computer vision project that is implemented based off of Christoph Lampert's paper in matlab. Given SIFT and HoG descriptors of specific attributes from 40 different training animal classes, the project aimed to classify 10 unknown animal test cases that contain the different attributes trained from the prior classes. The result was a 25% accuracy in successfully classifying new animals based on SIFT and HoG trained attributes. Accuracy will likely improve if I had used more features for each classes.

Pedestrian Detection

This is another computer vision project that uses HoG features of pedestrians and classifies through SVM training. The implementation is based on the paper by Navneet Dalal and Bill Triggs. The procedure involves training fixed frame sizes of pedestrians and scan over a given image searching for matching pedestrian frames.

Scene Categorization

Another computer vision project that uses SIFT features to categorize given scenes of kitchens, bathrooms, bedrooms given training images and test images. This implementation is based on the paper by Svetlana Lazebnik which utilizes a spatial pyramid representation to split the given scene into multiple sections and extract features with differential weights based on each section.

Video Frame Search

Using a bag of words approach and SIFT descriptors, this project searches through a movie for matching area of a frame with other frames within the movie.

Feature Extraction

Content Aware Resizing

This project aimed to resize the image through seam carving pixels with low energy, creating resized image that keeps the contents of the image as described in Shai Avidan's paper.

Simple Two-Level File System

This project utilized FUSE, an open source linux kernel extension to implement a user space program that allows for a simple two level file system. The resulting file system can thus call: stat, mkdir, ls, write, read, and unlink (delete).
